# Cachito Development Notes

## Animation Guidance

- Transforms are a web animator's best friend.
- Use transforms to move, scale, and rotate elements whenever possible.
- Use opacity for fades whenever possible.
- Prefer transforms and opacity over layout properties like `top`, `left`, or `margin` for animation.
- Transforms and opacity are more performant because they do not affect layout, which reduces browser work and usually produces smoother animations.
